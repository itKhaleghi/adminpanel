معرفی پروژه

پنل ادمین Itkhaleghi یک پنل مدیریتی کامل و واکنش‌گرا است که با HTML5, CSS3 و JavaScript توسعه داده شده است. این پنل مناسب برای سیستم‌های مدیریت محتوا، پنل‌های مدیریتی و برنامه‌های تحت وب است.

واکنش‌گرا

طراحی کاملاً ریسپانسیو برای تمام دستگاه‌ها از موبایل تا دسکتاپ

دارک مود

پشتیبانی از حالت تاریک و روشن با قابلیت تغییر توسط کاربر

پرسرعت

بهینه‌شده برای عملکرد بالا و تجربه کاربری روان

راهنمای توسعه

ساختار فایل‌ها

project/ ├── index.html # صفحه اصلی پنل ├── document.html # صفحه مستندات ├── blank.html # صفحه خالی ├── login.html # صفحه ورود ├── register.html # صفحه ثبت نام ├── 404.html # صفحه خطای 404 ├── 403.html # صفحه خطای 403 ├── css/ │ └── style.css # فایل استایل اصلی ├── js/ │ └── main.js # فایل JavaScript اصلی └── images/ # پوشه تصاویر

راه‌اندازی اولیه

برای شروع کار با پنل، کافیست فایل index.html را در مرورگر باز کنید. هیچ وابستگی خارجی به جز فونت آویکون ندارد.

اضافه کردن صفحه جدید

برای اضافه کردن صفحه جدید:

  1. یک فایل HTML جدید ایجاد کنید (مثال: products.html)
  2. ساختار اصلی را از index.html کپی کنید
  3. محتوا را در بخش main-content قرار دهید
  4. لینک صفحه جدید را به منوی sidebar اضافه کنید

استایل‌دهی و تم‌ها

متغیرهای CSS

تم اصلی با استفاده از متغیرهای CSS طراحی شده است:

:root { --color-main: #396aff; --color-main-dark: #2a52d1; --bg: #fbfefd; --bg2: #dce5ff; --main-accent: #e9eefd; --shadow: 0 0 16px rgba(17, 17, 26, .1); --color-text: #333; --color-text-light: #666; } .dark-mode { --bg: #1a1a2e; --color-text: #f1f1f1; --bg2: #16213e; --main-accent: #0f3460; }

کلاس‌های کمکی

کلاس توضیحات
.btn دکمه اصلی با رنگ آبی
.btn-secondary دکمه ثانویه با رنگ خاکستری
.w-full عرض 100%
.stat-card کارت آمار و اطلاعات

کامپوننت‌های اصلی

منوی کناری

منوی کناری با کلاس .sidebar قابل کنترل است. برای جمع کردن منو از کلاس .collapsed استفاده کنید.

جدول داده‌ها

برای ایجاد جدول از ساختار زیر استفاده کنید:

<div class="table-container"> <table class="data-table"> <thead> <tr> <th>عنوان ستون</th> ... </tr> </thead> <tbody> <tr> <td>مقدار</td> ... </tr> </tbody> </table> </div>

سیستم تب‌ها

برای ایجاد تب‌ها از ساختار زیر استفاده کنید:

<div class="content-tabs"> <div class="tabs-header"> <button class="tab-btn active" data-tab="tab1">تب 1</button> <button class="tab-btn" data-tab="tab2">تب 2</button> </div> <div class="tabs-content"> <div id="tab1" class="tab-pane active">محتوا 1</div> <div id="tab2" class="tab-pane">محتوا 2</div> </div> </div>
